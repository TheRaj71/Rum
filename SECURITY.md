# Security Policy

## Reporting Security Vulnerabilities

If you discover a security vulnerability in this project, please report it responsibly. Do not create public issues or pull requests for security vulnerabilities.

To report a security issue, please email the maintainer at theraj05@duck.com or create a private issue on GitHub.

We will acknowledge your report within 48 hours and provide a more detailed response within 7 days indicating our next steps.

Thank you for helping keep this project secure.
